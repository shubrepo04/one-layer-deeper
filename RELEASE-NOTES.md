# Release Notes

## v1.0.0

Initial release of the `one-layer-deeper` skill.

**What's included:**
- Core skill prompt with 5 recognition patterns (wrong problem, wrong solution, systemic issues, baked-in assumptions, second-order effects)
- Depth calibration guide (shallow / target / too deep)
- Language patterns for surfacing reframes without being preachy
- Anti-patterns section to prevent overcorrection
- Common scenario library: technical, management, strategy, personal decisions, analysis
